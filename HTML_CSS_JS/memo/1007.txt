반응형 홈페이지 포트폴리오 선정

창작 금지 (영역 제거, 다른 홈페이지 참고해서 영역 변경)
처음 제작은 난이도 쉬운거 권장
디자인 o

반응형 레이아웃 플러그인 x (예 : 투썸플레이스)

이메일, 카톡, 문자 홈페이지 주소 2개 이상 보내기

word-break : keep-all;
자간을 줄이지 않고 단어에 맞춰서 줄을 바꿈


$('선택자').toggle();
선택자가 화면에 보이면 hide를, 화면에 안보이면 show를 함
클릭 이벤트에만 사용 가능

자바스크립트, 제이쿼리 명령어는 인라인 스타일로 적용됨

$('선택자').parent(); : 선택자의 부모를 선택함
$('선택자').siblings(); : 선택한 요소의 형제 요소를 모두 선택함
$('선택자').children(); : 선택한 요소의 자식 요소를 모두 선택함(건너 자식은 선택 x)
$('선택자').find() : 선택자의 자식 요소를 선택함(건너 자식도 선택 o)
