var 변수 선언 : 중간에 값 변경 o
const 변수(상수) 선언 : 중간에 값 변경 x

연결연산자 ,(콤마)
결합연산자 +

++num 선증가 : num변수값에 1을 더함
먼저 1을 더하고 =(대입연산자)를 나중에 실행함

num++ 후증가 : num변수값에 1을 더함
=(대입연산자)를 먼저 실행하고 1을 더함

&& and 연산자
조건이 모두 충족되면 true
(조건이 1개라도 충족되지 않으면 false)

|| or 연산자
조건이 1개라도 충족되면 true
(조건이 모두 충족되지 않으면 false)


로그인 알고리즘
1. 로그인 버튼을 누르면 뭔가를 하겠다
2. 손님이 input태그에 입력한 값을 가져와서 변수에 넣기
3. 손님이 입력한 값과 기록된 값이 일치하는지 and 연산자로 검사
4. if명령어를 이용해서 충족되면 로그인 성공, 충족되지 않으면 실패
