float : left
왼쪽으로 차곡 차곡 순서대로 정렬
공중에 뜸
float 적용된 요소의 밑에 아무도 없다면 신경 x
float 적용된 요소의 밑에 누가 있다면 레이아웃이 풀어짐
float 적용된 요소는 면적 o
★float 적용은 서로 형제 관계에 적용
모든 브라우저에서 문제 없이 작동 o (규모 큰 회사에서 주로 사용)

해결방법
1. float 적용된 윗부모에 높이 넣어서 해결(반응형에서 사용 x)
2. float 적용된 밑의 요소에 clear : both 적용
3. float 적용된 윗부모에 .clearfix:after(content : "", clear : both, display : block); 적용


블록요소 특징
1. 줄 바꿈
2. 기본값이 너비 100%
3. 면적 o


인라인 요소 특징
1. 면적 x
2. 줄바꿈 x
3. 기본 여백 o

인라인 블록 특징
1. 줄 바꿈 x
2. 면적 o
3. 기본 여백 o
4. 기본값이 너비 100% 아님
